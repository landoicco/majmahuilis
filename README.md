# Majmahuilis

A classic Doom like game made with Pygame.
To play, run:

``` shell
uv run src/main.py
```

What means *Majmahuilis*? [Take a look](https://gdn.iib.unam.mx/diccionario/mamahuiliz/25933)

The roots of this project are inspired on the amazing work of [Coder Space](https://youtu.be/ECqUrT7IdqQ?si=1NiPnt29hNCoSkMp)

## Tech notes

If FPS drop for no reason, try cleaning the cache:

``` shell
uv cache clean
```
